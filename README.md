# 拼豆图纸生成器

一个简单的像素画/拼豆图纸生成工具，可以将任意图片转换为像素风格的图纸。

## 功能特点

- 📷 图片上传与转换
- 🎨 自动颜色匹配与优化
- ✏️ 画笔、填充、吸管等编辑工具
- 🔍 颜色定位与高亮
- 📊 颜色统计与占比分析
- 📱 移动端适配
- 💾 导出高清图纸

## 快速开始

### 安装依赖

```bash
npm install
```

### 启动开发服务器

```bash
npm start
```

### 构建生产版本

```bash
npm run build
```

## 使用说明

1. 点击"上传"按钮选择图片
2. 设置图纸宽度（格子数量）
3. 系统自动将图片转换为像素图纸
4. 使用工具栏进行编辑：
   - 画笔：点击或拖动修改颜色
   - 填充：一键填充相邻同色区域
   - 吸管：吸取画布上的颜色
5. 点击"优化"可过滤占比过小的颜色
6. 点击"导出"保存图纸
